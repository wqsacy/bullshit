<h1 align="center"> bullshit </h1>

<p align="center"> 狗屁不通文章生成器PHP版.</p>

## Installing

```shell
$ composer require wangqs/bullshit -vvv
```

## Usage

```
use Wangqs\Bullshit;
$bullshit = new Bullshit();
echo $bullshit::generator("hello word",1000);
```
